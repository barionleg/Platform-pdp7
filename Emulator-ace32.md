## About
**Developement state:** :red_circle: 2010

**ACE32** is a emulator for **Jupiter Cantab - Jupiter Ace** made by **Paul Robson** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.5 (x86)**, released on **2010-02-13**, it runs on **Windows XP**, you can start this emulator with **ACE32XP.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_ACE 32 - A Dos emulator for the Jupiter ACE._

[:link: **Emulator website**](http://www.jupiter-ace.co.uk/ace32.html)
