## About
**Developement state:** :red_circle: 2006

**Dapple** is a emulator for **Apple 2** made by **Steve Nickolas** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.42_v1.5a3 (x86)**, released on **2006-11-30**, it runs on **Windows XP,Vista**, you can start this emulator with **dap.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_Apple is an Apple ][ emulator that aims to support a well-fitted Apple ][, ][+ or //e machine with 64K. It is based on Marat Fayzullin's M6502 with 65C02 additions and Marcel de Kogel's Z80Em._

[:link: **Emulator website**](https://sourceforge.net/projects/dapple/)
