## About
**Developement state:** :red_circle: 2000

**DVE** is a emulator for **GCE/MB Vectrex** made by **Keith Wilkins** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.0b12-dos (x86)**, released on **2000-02-01**, it runs on **DOS 6.22**, you can start this emulator with **VECTREX.EXE**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_DVE (DOS Vectrex Emulator) is a GCE Vectrex emulator for DOS._

[:link: **Emulator website**](http://www.atarihq.com/vectrex/dve.htm)
