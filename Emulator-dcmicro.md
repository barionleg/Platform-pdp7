## About
**DCMicroColor** is a emulator for **Matra/Hachette Alice** made by **Daniel Coulom** and has a **Freeware** license.

The latest version of the emulator is **v1.0**, released on **2008-08-06**, it runs on **Windows XP,Vista,7,8,10 (x86,x64)**, you can start this emulator with **dcmicrocolor.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_DCMicroColor is a Tandy MC10 and Matra-Hachette Alice 4K emulator._

[**Emulator website**](http://alice32.free.fr/)
