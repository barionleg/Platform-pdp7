## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/kat5200_logo.jpg "kat5200 logo.")

**Developement state:** :large_blue_circle: 2017

**kat5200** is a emulator for **Atari 5200** made by **Brian Berlin** and has a **GNU GPL** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.7.1 (x86)**, released on **2017-02-18**, it runs on **Windows Vista,7,8,10**, you can start this emulator with **kat5200.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_kat5200 is an emulator for the Atari 5200 and 8-bit computers._

[:link: **Emulator website**](http://kat5200.jillybunch.com/)
