## About
**Developement state:** :large_blue_circle: 2017

**winTVC** is a emulator for **Videoton TV-Computer** made by **Gr√≥sz Attila** and has a **Freeware** license, the GUI language is **Hungarian**.

The latest version of the emulator on EDC is **1.1.0-win64 (x64)**, released on **2017-02-01**, it runs on **Windows 98,XP,Vista,7,8**, you can start this emulator with **WinTVCx64.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_WinTVC is my Videoton TV Computer (TVC) emulator for Windows. The TVC is very similar to the Enterprise 128._

[:link: **Emulator website**](http://gaia.atilia.eu/content/view/2/5/)
