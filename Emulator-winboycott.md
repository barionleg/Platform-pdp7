## About
**Developement state:** :red_circle: 2000

**WinBoycott** is a emulator for **Nintendo Game Boy, Nintendo Game Boy Color** made by **Julien FRELAT aka Gollum** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.63b (x86)**, released on **2000-03-13**, it runs on **Windows 95,98,XP**, you can start this emulator with **WinBoycott.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_WinBoycott has a high compatibility rate and reasonable speed. This emulator is however lacking a full-screen mode and any type of hardware acceleration. Classic GameBoy, Super GameBoy and GameBoy Color are all supported by this emulator. This is a port of the DOS-based GameBoy emulator Boycott._

[:link: **Emulator website**](http://boycott.emuunlim.com/)
