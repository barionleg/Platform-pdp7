## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/em7_logo_200.jpg "EM-7 logo.")

**Developement state:** :red_circle: 1998

**EM-7** is a emulator for **Fujitsu FM-7** made by **Yoshimi** and has a **Freeware** license, the GUI language is **Japanese**.

The latest version of the emulator on EDC is **2.2b-dos (x86)**, released on **1998-10-21**, it runs on **DOS 6.22**, you can start this emulator with **EM7SWUP.EXE**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_EM7 is an emulator for the Fujitsu FM-7. NOTE: Emulator and Website are in Japanese._

[:link: **Emulator website**](http://www23.tok2.com/home/fm7emu/ysm4/ysm44.htm)
