## About
**Developement state:** :large_blue_circle: 2016

**MZ700WIN** is a emulator for **Sharp MZ-700/1500** made by **Takeshi Maruyama (marukun)** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.56 (x86)**, released on **2016-09-12**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **mz700win.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_mz700win - an MZ-700 emulator._

[:link: **Emulator website**](http://www.retropc.net/mz-memories/mz700/)
