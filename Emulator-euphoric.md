## About
**Developement state:** :white_circle: 2014

**Euphoric** is a emulator for **Oric Atmos/Telestrat** made by **Fabrice Franc√®s** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **Build-1019 (x86)**, released on **2014-11-21**, it runs on **Windows XP,Vista**, you can start this emulator with **euphoric.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_Euphoric is a Dos/Windows emulator for the Oric computer_

[:link: **Emulator website**](http://oric.free.fr/emulator.html)
