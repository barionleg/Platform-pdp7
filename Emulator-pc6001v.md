## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/pc6001v_logo_200.jpg "PC6001V logo.")

**Developement state:** :large_blue_circle: 2016

**PC6001V** is a emulator for **NEC PC-6001** made by **Yumitaro** and has a **GNU GPLv2.1** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.25-win64 (x64)**, released on **2016-08-16**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **pc6001v.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_PC6001V is an NEC PC-6000 series emulator. NOTE: Website is in Japanese._

[:link: **Emulator website**](http://papicom.net)
