## About
**Developement state:** :red_circle: 1995

**ApplePC** is a emulator for **Apple 2** made by **David Ellsworth** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.52 (x86)**, released on **1995-06-13**, it runs on **DOS 6.22**, you can start this emulator with **APPLE.EXE**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Apple //c Emulator._

[:link: **Emulator website**](ftp://users.aol.com/davidells/ApplePC/)
