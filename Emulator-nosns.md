## About
**no$sns** is a emulator for **Nintendo Super Nintendo** made by **Martin Korth** and has a **Freeware** license.

The latest version of the emulator on EDC is **1.6 (x86,x64)**, released on **2016-03-21**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **NO$SNS.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_nocash SNES/Super Famicom emulator/debugger for windows._
