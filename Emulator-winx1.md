## About
**Developement state:** :red_circle: 2000

**WinX1** is a emulator for **Sharp X1** made by **Kenjo** and has a **Freeware** license, the GUI language is **Japanese**.

The latest version of the emulator on EDC is **0.08 (x86)**, released on **2000-02-05**, it runs on **Windows 98,XP**, you can start this emulator with **winx1.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_WinX1 is a Sharp X-1 series emulator._

[:link: **Emulator website**](http://web.archive.org/web/20011216045650/http://www.retropc.net/kenjo/winx1.html)
