## About
**Developement state:** :red_circle: 2008

**QL2K** is a emulator for **Sinclair QL** made by **Jimmy Montesinos** and has a **Postcardware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.01-win64 (x64)**, released on **2008-11-15**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **QL2K-x64.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_meisei is an MSX emulator. NOTE: BIOS is NOT required but will improve emulation and allow access to extra features._

[:link: **Emulator website**](http://www.jadiam.org/QL/)
