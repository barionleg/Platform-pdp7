## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/joyce_logo.jpg "JOYCE logo.")

**Developement state:** :large_blue_circle: 2017

**JOYCE** is a emulator for **Amstrad PCW/Schneider Joyce** made by **John Elliott** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.2.11 (x86)**, released on **2017-02-13**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **xjoyce.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_JOYCE emulates a PCW8000 series computer. Run it, and your PC becomes a PCW._

[:link: **Emulator website**](http://www.seasip.info/Unix/Joyce/download.html)
