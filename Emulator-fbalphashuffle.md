## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/fbalphashuffle_logo.jpg "FinalBurn Alpha Shuffle logo.")

**Developement state:** :large_blue_circle: 2016

**FinalBurn Alpha Shuffle** is a emulator for **Coleco ColecoVision, Capcom Play System 1, Capcom Play System 2, Capcom Play System 3, Sega Mega Drive/Genesis, Sega Game Gear, MSX Home Computer, SNK NeoGeo, NEC Turbografx-16/PC-Engine, NEC Supergrafx/PC-Engine 2, IGS PolyGameMaster, Sega System 16, Sega System 18, Sega SG-1000, Sega Master System** made by **KOF2112, Leaf, Creamymami** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.4.0-win64 (x64)**, released on **2016-03-13**, it runs on **Windows Vista,7,8,10**, you can start this emulator with **Fbas-x64.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_FB Alpha Shuffle is an unofficial version based on FBA with many features._

[:link: **Emulator website**](http://code.google.com/p/fbashuffle/)
