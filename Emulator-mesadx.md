## About
**MesaDX** is a emulator for **Sega Game Gear** made by **Eric R Quinn** and has a **Freeware - Non-Commercial** license.

The latest version of the emulator on EDC is **0.04 (x86)**, released on **2006-09-23** :triangular_flag_on_post:, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **MesaDX.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_MesaDX is a Sega Master System and Game Gear emulator._
