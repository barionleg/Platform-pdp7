## About
**Developement state:** :large_blue_circle: 2017

**Ootake** is a emulator for **NEC Turbografx-16/PC-Engine** made by **Kitao Nakamura** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.80 (x86)**, released on **2017-04-16**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Ootake.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_PC Engine (Turbo Grafx 16) emulator since 2006/07_

[:link: **Emulator website**](http://www.ouma.jp/ootake/)
