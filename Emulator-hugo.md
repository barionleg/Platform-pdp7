## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/hugo_logo_200.jpg "Hu-Go! logo.")

**Developement state:** :red_circle: 2005

**Hu-Go!** is a emulator for **NEC Turbografx-16/PC-Engine, NEC PC-Engine CD** made by **Zeograd** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.12 (x86)**, released on **2005-04-09**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **hugo.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Good pce emulator. Plays sounds, support fullscreen. Plays most roms!_

[:link: **Emulator website**](http://www.zeograd.com)
