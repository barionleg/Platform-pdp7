## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/wataroo_logo.jpg "Wataroo logo.")

**Developement state:** :large_blue_circle: 2017

**Wataroo** is a emulator for **Watara Supervision** made by **Osman Celimli** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.7.0.2 (x86)**, released on **2017-02-26**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Wataroo.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Wataroo is a Watara SuperVision emulator. It was written after I lost a bet with Brandon Cobb of Super Fighter Team. As with a genuine SuperVision, the best way to use this software is not at all._

[:link: **Emulator website**](http://tailchao.com/Wataroo/index.php)
