## About
**Developement state:** :red_circle: 1997

**Free 64** is a emulator for **Commodore 64** made by **Brad Martin** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.01 (x86)**, released on **1997-09-08**, it runs on **Windows 95,XP**, you can start this emulator with **FREE64.EXE**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_Free 64 emulates the Commodore 64._

[:link: **Emulator website**](http://home.gvi.net/~martin/)
