## About
**Developement state:** :red_circle: 2006

**Virtual NEC Trek** is a emulator for **NEC PC-6001** made by **James the Animal Tamer** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.4a (x86)**, released on **2006-03-22**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **vnectrek.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Virtual NEC Trek, an emulator for the NEC Trek (aka NEC PC-6001A)._

[:link: **Emulator website**](http://www.geocities.ws/emucompboy/)
