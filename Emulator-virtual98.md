## About
**Developement state:** :red_circle: 1999

**Virtual98** is a emulator for **NEC PC-9801** made by **SOLTIN software** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.28 (x86)**, released on **1999-12-14**, it runs on **Windows 95,98**, you can start this emulator with **VIRTUAL9.EXE**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_This PC98 emulator requires MMX._

[:link: **Emulator website**](http://hwbb.gyao.ne.jp/soltin/virtual98.html)
