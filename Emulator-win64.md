## About
**Developement state:** :red_circle: 1998

**Win64** is a emulator for **Commodore 64** made by **Ralph Mason** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.40b (x64)**, released on **1998-02-22**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Win64.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_Win64 emulates the Commodore 64._

[:link: **Emulator website**](http://www.geocities.com/SilliconValley/Park/6558)
