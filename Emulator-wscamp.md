## About
**WSCamp** is a emulator for **Bandai Wonderswan** made by **UNKNOWN** and has a **UNKNOWN** license.

The latest version of the emulator on EDC is **0.21 (x86,x64)**, released on **2002-10-14** :triangular_flag_on_post:, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **WSCamp.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_WSCamp is a Bandai WonderSwan and WonderSwan Color emulator._
