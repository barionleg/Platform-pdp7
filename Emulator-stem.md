## About
**Developement state:** :red_circle: 2000

**STEM** is a emulator for **RCA Studio II** made by **Paul Robson** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2000-09-17 (x86)**, released on **2000-09-18**, it runs on **Windows 98,XP**, you can start this emulator with **WINSTEM.EXE**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_RCA Studio 2 emulator written in C and assembler._

[:link: **Emulator website**](http://www.robsons.org.uk/archive/www.classicgaming.com/studio2/index.htm)
