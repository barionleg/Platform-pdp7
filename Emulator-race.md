## About
**Developement state:** :red_circle: 2006

**RACE!** is a emulator for **SNK NeoGeo Pocket Color** made by **Judge_** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2006-06-13-source (x86)**, released on **2006-05-19**, it runs on ** **, you can start this emulator with **main.cpp**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_ACE! is a NGPC emulator originally envisioned for the GP2X.  It is currently available in binary form for GP2X, PC, and GP32.  It should be easily portable to other systems like PSP, Zodiac, DreamCast, etc._

[:link: **Emulator website**](https://sourceforge.net/projects/koyote-land/)
