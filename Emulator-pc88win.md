## About
**Developement state:** :red_circle: 1999

**PC88Win** is a emulator for **NEC PC-8801** made by **SOLTIN software** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.14 (x64)**, released on **1999-03-05**, it runs on **Windows 95,98**, you can start this emulator with **PC88WIN.EXE**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_a NEC - PC8801 Emulator on the Windows platform._

[:link: **Emulator website**](http://hwbb.gyao.ne.jp/soltin/pc88win.html)
