## About
**Red Dragon** is a emulator for **Nintendo Virtual Boy** made by **frostgiant and Parasyte** and has a **Freeware** license.

The latest version of the emulator on EDC is **0.38 (x86)**, released on **2004-09-21** :triangular_flag_on_post:, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **rdragon.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Red Dragon is a Nintendo Virtual Boy emulator._
