## About
**Developement state:** :large_blue_circle: 2017

**Sharp 80** is a emulator for **Tandy/Radio Shack TRS-80 Color Computer** made by **Matthew Hamilton** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.1.0.99 (x86)**, released on **2017-04-20**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Sharp80.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_This is a full-featured emulator of a TRS-80 Model III microcomputer. It is free of charge and all source code is publically available._

[:link: **Emulator website**](http://www.sharp80.com/)
