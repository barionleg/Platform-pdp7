## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/quasi88_logo.jpg "QUASI88 logo.")

**Developement state:** :red_circle: 2013

**QUASI88** is a emulator for **NEC PC-8801** made by **S. Fukunaga** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.6.4 (x86)**, released on **2013-03-26**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **QUASI88win32.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_Portable PC-8801 emulator for Unix/X11, Mac OS X, and Windows._

[:link: **Emulator website**](http://www.eonet.ne.jp/~showtime/quasi88/)
