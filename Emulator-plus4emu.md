## About
**Developement state:** :red_circle: 2008

**plus4emu** is a emulator for **Commodore 16/plus4** made by **Istvan Varga** and has a **GNU GPLv2** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.2.9.2 (x86)**, released on **2008-09-26**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **plus4emu.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_plus4emu is an open source, portable emulator of the Commodore 264 family of computers (C16, C116, and Plus/4), written in C++, and supporting Windows and POSIX platforms (32 bit Windows, 32 and 64 bit Linux, and MacOS X have been tested). It implements accurate, high quality hardware emulation, however, the system requirements are higher than that of most other emulators_

[:link: **Emulator website**](http://plus4emu.sourceforge.net/)
