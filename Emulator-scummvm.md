## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/scummvm_logo_200.jpg "ScummVM logo.")

**Developement state:** :large_blue_circle: 2016

**ScummVM** is a emulator for **Scumm Virtual Machine** made by **ScummVM Team** and has a **Open Source** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.9.0 (x86)**, released on **2016-10-19**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **scummvm.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_ScummVM is a program which allows you to run certain classic graphical point-and-click adventure games, provided you already have their data files. The clever part about this: ScummVM just replaces the executables shipped with the games, allowing you to play them on systems for which they were never designed!_

[:link: **Emulator website**](https://www.scummvm.org)
