## About
**Developement state:** :large_blue_circle: 2017

**hhugboy** is a emulator for **Nintendo Game Boy, Nintendo Game Boy Color** made by **Taizou** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.1.8 (x86)**, released on **2017-01-07**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **hhugboy.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_a Game Boy/Game Boy Color emulator for Windows._

[:link: **Emulator website**](http://hhug.me)
