## About
**Developement state:** :red_circle: 1999

**W88** is a emulator for **NEC PC-8801** made by **anon** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.05 (x86)**, released on **1999-12-14**, it runs on **Windows 95,98**, you can start this emulator with **W88.EXE**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_PC98 emulator for windows._

[:link: **Emulator website**](http://www.cug.net/~anonB/w88/)
