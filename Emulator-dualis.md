## About
**Dualis** is a emulator for **Nintendo DS** made by **Mic** and has a **UNKNOWN** license.

The latest version of the emulator on EDC is **20.4 (x86,x64)**, released on **2007-05-27** :triangular_flag_on_post:, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **dualis.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Dualis is a Nintendo DS emulator._
