## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/nanowasp_logo_200.jpg "NanoWasp logo.")

**Developement state:** :red_circle: 2007

**NanoWasp** is a emulator for **Applied Technologies Microbee** made by **Dave Chuchill** and has a **GNU GPLv3** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2.0b (x86)**, released on **2007-08-05**, it runs on **Windows Vista,7,8,10**, you can start this emulator with **nanowasp.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_An emulator for the Microbee, a microcomputer popular in Australia in the early '80s._

[:link: **Emulator website**](http://www.stairwaytohell.com/atom/wouterras/)
