## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/swfopener_logo_200.jpg "SWF Opener logo.")

**Developement state:** :red_circle: 2005

**SWF Opener** is a emulator for **Adobe Flash Games** made by **UnH Solutions** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.3 (x86)**, released on **2005-04-27**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **SWFOpener.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_SWF Opener requires macromedia flash player (flash.ocx)_

[:link: **Emulator website**](http://www.browsertools.net/SWF-Opener)
