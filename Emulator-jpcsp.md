## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/jpcsp_logo.jpg "JPCSP logo.")

**Developement state:** :red_circle: 2010

**JPCSP** is a emulator for **Sony PlayStation Portable** made by **JCPSP Team** and has a **GNU GPLv3** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.6-win64 (x64)**, released on **2010-09-19**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **start-windows-amd64.bat**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_JPCSP is a Sony Playstation Portable emulator. NOTE: You may have to amend the '.bat' file to point to the correct location of your Java installation._

[:link: **Emulator website**](http://www.jpcsp.org)
