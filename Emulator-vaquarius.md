## About
**Virtual Aquarius** is a emulator for **Mattel Aquarius** made by **James The Animal Tamer** and has a **UNKNOWN** license.

The latest version of the emulator is **v0.72a**, released on **2008-08-05**, it runs on **Windows XP,Vista,7,8,10 (x86,x64)**, you can start this emulator with **aquarius.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Virtual Aquarius is an emulator for the Radofin Electronics Aquarius Home Computer System._

[**Emulator website**](http://www.lchr.org/a/18/2t/)
