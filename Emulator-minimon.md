## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/minimon_logo.jpg "Minimon logo.")

**Developement state:** :red_circle: 2005

**Minimon** is a emulator for **Nintendo Pok√©mon mini** made by **Bryon Vandiver** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **2005-01-20 (x86)**, released on **2005-01-20**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Minimon.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_This is the world first Pokemon Mini emulator._

[:link: **Emulator website**](http://www.sublab.net/minimon/minimon.html)
