## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/luagb_logo.jpg "LuaGB logo.")

**Developement state:** :large_blue_circle: 2017

**LuaGB** is a emulator for **Nintendo Game Boy, Nintendo Game Boy Color** made by **Nicholas Flynt** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.1.0-alpha-win (x86)**, released on **2017-04-09**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **LuaGB.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_A gameboy emulator written in Pure Lua. Approaching feature completeness, but still a work in progress. This is designed to be fairly cross platform, and currently consists of a platform-independent gameboy module which contains the emulator, and a Love2D interface, presently the only supported platform._

[:link: **Emulator website**](https://github.com/zeta0134/LuaGB)
