## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/kcemu_logo_200.jpg "KCemu logo.")

**Developement state:** :red_circle: 2010

**KCemu** is a emulator for **Robotron KC series** made by **Paul Torsten** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.5.1 (x86)**, released on **2010-03-07**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **bin\kcemu.exe**.

:warning: _**Please note:**_ This emulator **needs BIOS image(s)** to start ROM files!

_The KC85 connects to the Internet and can access USB storage media. This is made possible by the new module M052. This includes both a network interface and a USB host port._

[:link: **Emulator website**](http://kcemu.sourceforge.net/kcemu/)
