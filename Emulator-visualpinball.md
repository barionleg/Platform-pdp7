## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/visualpinball_logo.jpg "Visual Pinball logo.")

**Developement state:** :large_blue_circle: 2016

**Visual Pinball** is a emulator for **Pinball - Visual Pinball** made by **Randy Davis** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **10.1 (x86)**, released on **2016-06-07**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **VPinballX.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Visual Pinball can more or less faithfully reproduce the gameplay of most pinball tables created before 1990.  More modern tables tend to use complicated engineering which is beyond that supported by VP.  Of course, every table has it's own gimmicks, which usually can be reproduced with varying degrees of success._

[:link: **Emulator website**](http://www.randydavis.com/vp/)
