## About
**Developement state:** :red_circle: 2001

**GameLad** is a emulator for **Nintendo Game Boy, Nintendo Game Boy Color** made by **Torbjörn Söderstedt** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.6.1 (x86)**, released on **2001-10-05**, it runs on **Windows 95,98,XP**, you can start this emulator with **Game Lad.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_GameLad is an emulator aimed at developers. This emulator is equipped with both a debugger and a disassembler and has the ability to throttle the emulation process on the fly. It also has the ability to run multiple virtual GameBoys at once. This emulator requires Direct X 8.0 or better and has link cable support._
