## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/cogwheel_logo.jpg "Cogwheel logo.")

**Developement state:** :red_circle: 2009

**Cogwheel** is a emulator for **Coleco ColecoVision, Sega Game Gear, Sega SG-1000, Sega Master System** made by **Ben Ryves** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **1.0.3.0beta3 (x86)**, released on **2009-08-25**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Cogwheel Interface.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_A software emulator primarily for 8-bit Sega hardware, such as the Sega Master System, Sega Game Gear and SG-1000. The ColecoVision is also supported._

[:link: **Emulator website**](http://benryves.com/products/cogwheel)
