## About
**NeoChip8** is a emulator for **Cosmac VIP** made by **Eric Scrivner aka zenogais** and has a **UNKNOWN** license.

The latest version of the emulator on EDC is **2004.06.30 (x86,x64)**, released on **2004-06-30 :triangular_flag_on_post: **, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **NeoChip8.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_NeoChip8 is a RCA COSMAC VIP and Chip-8 emulator._
