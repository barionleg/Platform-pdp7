## About
**Developement state:** :large_blue_circle: 2017

**April NES Emulator** is a emulator for **Nintendo NES/Famicom** made by **erspicu_brox** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **20170224 (x86)**, released on **2017-02-24**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **AprNes.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_AprNes Nintendo Red and White Machine Simulator._

[:link: **Emulator website**](http://www.baxermux.byethost18.com/myemu/AprNes/index.htm?i=1)
