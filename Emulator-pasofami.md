## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/pasofami_logo.jpg "Pasofami logo.")

**Developement state:** :large_blue_circle: 2017

**Pasofami** is a emulator for **Nintendo NES/Famicom** made by **N. Andou** and has a **Freeware** license, the GUI language is **Japanese**.

The latest version of the emulator on EDC is **2.32 (x86)**, released on **2017-04-08**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **pasofami.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Pasofami is a Nintendo Entertainment / Nintendo Famicom emulator and debugger for Windows. Originally making it's debut in 1995 Pasofami was one of the first Famicom emulators to come out of Japan and over time has developed a rich feature set._

[:link: **Emulator website**](http://hp.vector.co.jp/authors/VA005758/)
