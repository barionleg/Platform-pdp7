## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/visualboyadvancerr_logo.jpg "VisualBoyAdvance ReRecording logo.")

**Developement state:** :red_circle: 2013

**VisualBoyAdvance ReRecording** is a emulator for **Nintendo Game Boy, Nintendo Game Boy Advance, Nintendo Game Boy Color** made by **adelikat, DarkKobold, gocha, klmz, nitsuja, P.JBoy** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **v24-svn480 (x86)**, released on **2013-04-15**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **vba-v24m-svn480.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_VBA-ReRecording is the re-recording branch of the stable 1.7.2 version of VisualBoyAdvance, The primary function of this branch is to expand features related to the creation of Tool-assisted movies._

[:link: **Emulator website**](https://code.google.com/archive/p/vba-rerecording/)
