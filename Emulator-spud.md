## About
**Developement state:** :large_blue_circle: 2015

**Spud** is a emulator for **Sinclair ZX Spectrum** made by **Richard Chandler** and has a **GNU GPLv2** license, the GUI language is **English**.

The latest version of the emulator on EDC is **0.240 (x86)**, released on **2015-02-08**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **Spud.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_Spud is a ZX Spectrum 48k/128k Emulator written in C++, it emulates 16K/48K/128K/+2/+2A/+3 machines._
