## About
![](https://github.com/PhoenixInteractiveNL/emuDownloadCenter/wiki/images_emulator/nemulator_logo.jpg "Nemulator logo.")

**Developement state:** :large_blue_circle: 2017

**Nemulator** is a emulator for **Nintendo NES/Famicom, Sega Master System, Sega Game Gear** made by **James Slepicka** and has a **Freeware** license, the GUI language is **English**.

The latest version of the emulator on EDC is **4.2 (x86)**, released on **2017-03-01**, it runs on **Windows XP,Vista,7,8,10**, you can start this emulator with **nemulator.exe**.

This emulator does **NOT** specificly need BIOS image(s) to start ROM files!

_nemulator is a Nintendo Entertainment Systems and Sega Master System emulator._

[:link: **Emulator website**](http://nemulator.com/)
